\section{自動微分とは}
\begin{frame}{問題意識}
 計算機上で微分（勾配ベクトル）を速く，正確に計算したい．
 微分の用途：
 \begin{enumerate}
  \item 最適化アルゴリズム
  \item 感度分析
  \item 物理モデリング
  \item 機械学習 etc...
 \end{enumerate}
 微分は極限を扱うが，計算機には「無限」が理解できない．
 → 近似の必要性
 よく見る微分の姿：
 \begin{enumerate}
  \item 数値微分: $\frac{df(x)}{dx} = \lim_{h \to 0} \frac{f(x+h)-f(x)}{h}$
  \item 数式微分: $\frac{d(u(x)+v(x))}{dx} = u(x)\frac{dv(x)}{dx} + \frac{du(x)}{dx}v(x)$
 \end{enumerate}
\end{frame}
